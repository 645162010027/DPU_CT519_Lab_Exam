ประโยชน์ที่ได้จากการเรียน วิชา CT519 ( Cloud Computer )

นักศึกษาได้รับความรู้เกี่ยวกับ เทคโนโลยี Cloud Computer ทั้งแบบ Public Cloud, Private Cloud, Hybrid Cloud และ
On Premise. เพื่อให้นักศึกษาสามารถนำความรู้ไปปรับใช้กับการทำงาน

ในรายวิชายังได้อธิบายองค์ประกอบของ Cloud Computer ไม่ว่าจะเป็นการทำ RAID, VM, Container และบริการ
IaaS, SaaS, PaaS รวมถึงการใช้งาน DAS, NAS, SAS

นักศึกษาได้ฝึกทำ Lab VM, GoLang, Docker เพื่อทบทวนและฝึกการทำระบบเพื่อให้เข้าใจและสามารถนำไปใช้งานได้
